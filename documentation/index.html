Introduction

Full Convert is a database conversion application. It enables you to easily copy contents of your database to a database in different format.

All your existing tables will be recreated in your desired target database and all of the data will be copied. Table indexes and relationships will be created, as well.

Advanced users can customize the conversion process in many ways - but we worked hard to make the straightforward database copy as easy as possible.

To start, take a look at the getting started guide. Once you are have completed the basic conversion, examine your target database to see that the data is there.

You can have conversion running repeatedly - many customers have it running at some point of time during the night, so that in the morning database is copies anew. Of course, you can change the table mapping so that target database is created differently on the fly. Tables can be renamed, columns can be renamed, data types can be changed - and much more.
